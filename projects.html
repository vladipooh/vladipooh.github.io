Advanced COVID-19 Detection Using Ensemble CNNs and Transfer Learning on Chest X-Rays

This work focuses on improving COVID-19 detection using chest X-ray images and advanced machine learning techniques. By developing a voting ensemble of pre-trained Convolutional Neural Networks (CNNs) and employing transfer learning, we addressed data insufficiency and enhanced classification accuracy. The study utilized datasets from COVID-19 image collections and the NIH Chest X-ray Dataset, applying data augmentation and heatmap analysis. The best-performing 3-voter ensemble model (ResNet18, ResNet34, AlexNet) achieved 90% accuracy, 82.5% recall, 97.1% precision, and an 89.19% F1-score, outperforming individual models.

Enhancing University Programme Recommendations Using MBTI, Gradient Boosted Trees, and Firefly Algorithm

In this study, a hybrid machine learning approach was developed to recommend university majors to students based on their MBTI personality type, using Gradient Boosted Trees and the Firefly Algorithm for feature selection. The research aimed to assist undecided students by analyzing their personal profiles, educational backgrounds, and environmental influences. The system significantly improved accuracy and reduced processing time compared to traditional methods, achieving 84.78% accuracy with the Firefly Algorithm.

Predicting Salaries of the Thai Labor Workforce Using Deep Learning and Feature Selection

In this research, we developed a salary prediction system for the Thai labor workforce using deep learning techniques. The study utilized a dataset of over 1.7 million users from a popular job search website, focusing on various personal and professional attributes such as age, work experience, educational background, and more. By applying deep neural networks (DNNs) combined with nature-inspired metaheuristic feature selection algorithms, we achieved an optimal R-squared value of 0.462 and a rapid runtime of 15.37 seconds. This research highlights the integration of advanced machine learning methods to improve prediction accuracy and efficiency in human resource management.
